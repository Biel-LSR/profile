## Portfolio Pessoal
Bem-vindo ao meu portfólio! Este projeto apresenta minhas habilidades e projetos como Desenvolvedor Web | Designer UI/UX.

### 🚀 Sobre o Projeto
Esta é uma página de portfólio pessoal desenvolvida para showcase de trabalhos e competências. O objetivo é fornecer uma visão clara e moderna sobre minha carreira e projetos, facilitando o 3contato com recrutadores e clientes.

### 🎯 Funcionalidades
Contato: links para redes sociais (LinkedIn, GitHub, etc.).
Design Responsivo: Layout adaptável para dispositivos móveis, tablets e desktops.

### 💻 Tecnologias Utilizadas
Frontend: HTML5, CSS3, JavaScript (ES6+)
Estilização: CSS Modules / Styled Components
Ferramentas: Git, GitHub
Hospedagem: GitHub Pages

Este projeto está configurado para deploy automático na GitHub Pages. A cada push para a branch main, o site é automaticamente atualizado.
